Research Project

Hallucination Detection with Fine‑Tuned Language Models

PyTorch · HuggingFace · LoRA · GPT‑2

Fine-tuned a GPT-2 classifier using LoRA to detect whether AI-generated responses were hallucinated or factual. Trained on the HaluEval dataset and improved classification accuracy from ~12% to 62%, demonstrating the impact of parameter-efficient fine-tuning on model reliability.

Research Project

EEG Data Analysis

Python · Data Visualization

Analyzed EEG datasets as part of a professor‑guided project. Cleaned and processed raw brain‑signal data, visualized patterns, and explored challenges in biomedical signal interpretation.

Research Direction

LLM Benchmarking for Rare Disease Diagnosis

GPT‑4o · GitHub Models · AI × Healthcare

Designed an evaluation framework to test whether LLM diagnostic accuracy changes across disease rarity tiers. This work explores how LLMs reason about rare conditions, where data scarcity and clinical ambiguity make diagnosis challenging. The goal is to understand model limitations and identify opportunities for safer, more reliable AI‑assisted clinical reasoning.
